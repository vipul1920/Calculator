**Overview**:
        This project is a simple calculator application developed in Java using the IntelliJ IDEA integrated development environment (IDE). 
        It provides basic arithmetic operations and serves as a foundational project for understanding Java.
        
**Features:**
        Basic Arithmetic Operations: Perform addition, subtraction, multiplication, and division.

**Getting Started:**
        **Java Development Kit(JDK)**: Ensure you have JDK installed on your system. You can download it from the (https://www.oracle.com/java/technologies/downloads/?er=221886)
        **IntelliJ IDEA**: This project was developed using IntelliJ IDEA. You can download it from the (https://www.jetbrains.com/idea/download/?section=windows)
        
**Installation:**
        **Clone the Repository**: Clone this repository to your local machine using  git clone https://github.com/vipul1920/Calculator.git
        Open in IntelliJ IDEA:
                              Launch IntelliJ Idea.
                              Navigate to File > Open and select the cloned repository folder.
        Build the Project:
                          Once the project is loaded navigate to Build > Run 'Calculator'. The calculator should appear in consoule window, allowing you to perform basic arthmetic operations.

**Project Structure:**
                  src/ : Contains the source code.
                  Calculator.java: Contains the logic for arithmetic operations.
